## 🤠 Welcome
This is a repo for teo's portfolio, currently hosted at [teo.fyi](https://teo.fyi).

The site is built as a gatsby project, with React.js / typescript / tailwind css


## To run the website locally

Clone the repo to `./portfolio`  and then 

```shell
cd portfolio
gatsby develop
```

The terminal will instruct you that you can now view the site at localhost:8000
